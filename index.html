Contenido Teórico

Los condicionales permiten que un programa tome decisiones según ciertas condiciones. Son fundamentales en programación para ejecutar instrucciones según diferentes escenarios.

Condicionales en Python: Ejemplos prácticos

Se ejecuta un bloque de código solo si se cumple una condición:

if condición:
    # código a ejecutar si la condición es True

Permite ejecutar un bloque si la condición es verdadera y otro si es falsa:

if condición:
    # si verdadero
else:
    # si falso

Permite evaluar varias condiciones en orden y ejecutar la primera que se cumpla:

if cond1:
    # acción1
elif cond2:
    # acción2
else:
    # acción por defecto

Puedes combinar condicionales dentro de otros condicionales para decisiones más complejas:

edad = 20
if edad >= 18:
    if edad < 21:
        print("Adulto joven")
    else:
        print("Adulto")
else:
    print("Menor de edad")
